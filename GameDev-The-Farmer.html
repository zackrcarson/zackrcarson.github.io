Personal Game Development Demo: The Farmer

This was a fun project I worked on similar to the popular game Stardew Valley. Try planting some crops (hit G to advance the game day quickly!), changing scenes (E to enter doors) breaking some rocks, following (Press T to advance the game time by 10 minutes!) or talking to the NPC's (hit enter to talk), cutting down some trees, or customizing your character (ESC to see pause menu, with save/quit/load/volume/character customization tabs). Have fun! The repository for this game can be found here.
