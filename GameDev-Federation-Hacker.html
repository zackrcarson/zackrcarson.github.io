Personal Game Development Demo: Federation Hacker

This is a simple terminal hacker game, where you can use your old-school computer to hack into the intergalactic colony personel records, the space ship flight records, or into the Solar Federation database. See if you can decrypt the encrypted anagram passwords in time! Try to beat all three difficulty levels if you can. The repository for this game can be found here.
